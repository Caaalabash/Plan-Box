## Plan-Box 基本流程

+ 新建一个`Sprint`, 作为一个任务周期, 那么需要有如下的数据

  + 名称

  + 描述

  + 开始时间

  + 结束时间

  + 总任务量

  + 完成任务量

  + 负责人

  + *各项Task

+ 各项`Task`, 作为一个任务周期中的一项关键任务, 那么需要有如下数据

  + 所属 `Sprint`

  + 任务优先级

  + 在对应的 `Sprint`中排序

  + 名称

  + 描述

  + 预估任务量

  + *负责人

  + 任务状态

  + *子任务

+ 各项`issue`, 作为一个`Task`中的子任务存在, 那么需要有如下的数据

  + 所属 `task`

  + 任务优先级

  + 名称

  + 描述

  + 预估任务量

  + 预估时间

  + 使用时间

  + 任务状态

  + 任务类型

  + *负责人
